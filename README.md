# :movie_camera: NgClips

Minimal video sharing app build in Angular

:link: [Demo](https://ng-clips-marius-paduraru.vercel.app/)

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm start
```

You can view the development server at `localhost:4200`.
